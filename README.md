# JobFlow Copilot

Dockerized FastAPI (API) + Next.js (web) starter.

## Run locally
1. Copy env: `cp .env.example .env` (Windows: copy the file manually)
2. Start: `docker compose up --build`
3. Open web: http://localhost:3000  |  API: http://localhost:8000/health
