# Professional Email Icons Reference

This document shows the professional Unicode symbols used in the ReminderEmail template instead of childish emojis.

## Main Symbols

| Symbol | Unicode | Usage | Description |
|--------|---------|-------|-------------|
| ◎ | U+25CE | Target/Bullseye | Main event icon, success footer |
| ◆ | U+25C6 | Diamond | Due date indicator, "Tomorrow" badge |
| ◷ | U+25F7 | Circle with dot | Time/clock indicator |
| ✦ | U+2726 | Star | AI-powered feature highlight |
| ▣ | U+25A3 | Square | Company/building icon |

## Design Principles

1. **Professional**: No childish emojis, only geometric Unicode symbols
2. **Email-Safe**: All symbols render consistently across email clients
3. **Sophisticated**: Clean, minimalist aesthetic matching dark theme
4. **Accessible**: High contrast and clear visual hierarchy
5. **Consistent**: Unified design language throughout
